## Babel 7

+ [Babel](https://babeljs.io/docs/en/)
+ [Using babel 7 with node](https://hackernoon.com/using-babel-7-with-node-7e401bc28b04)

#### Install

```
yarn add @babel/core @babel/cli @babel/preset-env @babel/polyfill
```
> babel core for compiler

```
yarn add @babel/node
```
> babel for node.js

#### Preset

```
"babel": {
  "presets": [
    "@babel/env"
  ]
}
```
> in packages
